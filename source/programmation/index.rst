Ecosystème
----------

L'ecosystème PostgreSQL dispose de nombreuses bibliothèques natives pour s'interfacer avec une multitude de langage.

Lorsque que pour un langage, aucune interface native n'est disponible, il est parfois possible de s'y connecter 
avec un driver ODBC.
